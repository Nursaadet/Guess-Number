## ✨ Features
- Prevents entering duplicate numbers
- Restricts input to numbers between 1–25
- Blocks text input
- Displays top-score indicator
- Uses DOM manipulation
- Saves data with Local Storage
- Restart game with “Again” button
- Visual feedback:
  - Green for win
  - Red for lose
